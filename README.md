# Portfolio Creation

**Overview**

A professional portfolio has been created as part of the Web Development class using HTML, CSS, Java Script and Bootstrap. It was a great learning, definitely need to mention the hands-on lab assignments helped a lot in correlating the theory and while working on the project in realtime.

- The webpage contains mainly 4 different sections - About | Work Experience | Projects | Contact.
- To meet the additional requirements, introduced a new section - Recognitions to implement accordion.
- I was excited to see how things would look if we have multiple pages, hence created experience.html.
- Features like form validation, smooth scrolling and better visbility via Mobile device has been implemented.
- The page has been created, tested and has been deployed via Github pages.

**Where to begin ?**

Go to the portfolio website and experience it( Link is in Journal), I would love to hear your feedback.
To dive into the code, please go to Github - find the repo and you could either download, clone, fork

**How to begin ?**

Login to github and on navigating to this repo, on top right click on CODE and it should list you the options we've doscussed above, you could choose to open the index.html after downloading and run the code to see it displaying my portfolio.

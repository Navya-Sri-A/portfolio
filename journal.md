**Portfolio Creation : Action plan**


Please find the related links here : 

* Link to the git public repo : https://github.com/Navya-Sri-A/portfolio

The step by step process for creating a professional portfolio using technologies like HTML, CSS, Java Script and Bootstrap

* ***Ideas phase :***
Though the requirements were pretty clearly outlined, I did brainstorm on how the layout should look like - do I want to create single or distribute the content into multiple page | Dark colors or light colors | Images or just colors for the background etc.

* ***Planning phase :***
As per the basic expectations the navigation bar should have ( About, Previous Work, Projects and Contact), can be single or multiple pages, I've began by creating single page and some basic styles and tried some colors as background.

* ***Prototype phase :***
On creating the basic layout, I felt the scope for a new page creation as I have previous work experience for 7 years it would be nice to have that documented it all. I began working on it, initially I struggled a bit in having the right spacing between the content since the padding from index.html created a huge padding on the experience.html page. 

* ***Project development phase :*** 
During this phase I was sure of having the baic layout lined up, decided to have experience.html and began working on css and contact form java script code.

* ***Testing phase :*** 
After multiple trial and error methods I've found few repetitive blocks of code in CSS and I've edited it all to make it better while accessing and improve readability.

* ***Improvements phase :***
Here is where I've noticed the outlook being decent and scope to improve the scrolling and mobile view perspectives and hence began working on it.

* ***Final phase :***
The final phase was to test the performance of the site, I've tried testing contact form options, scrolling, experience page navigation, mobile view perspective, etc, it all worked as expected and then I've began running WAVE evaluation tool to confirm any issues (Found a couple - Contrast errors and redundant links) fixed and ready to deploy the page.
